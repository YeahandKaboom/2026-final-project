# 💰 MONEY CRAFT GAME 💰

## 🚀 The Ultimate Mining Adventure! 🚀

### 🎮 GAME FEATURES:
- 🏘️ **Working Villages** - Villagers making tools, food, etc!
- 🚶 **Ground Walking** - Character walks on ground properly!
- 🕳️ **Mining Tunnels** - Press E to enter underground mines!
- 🛒️ **Working Shop** - Buy tools, weapons, items, Bitcoin upgrades!
- 💎 **Ruby Hills** - Sparkling ruby deposits!
- ₿ **Bitcoin Mines** - Crypto mining operations!
- 🌈 **Insane World Generation** - 21 biomes, 29 terrain types, 25 structures!
- 🐉 **Epic Boss Battles** - Ultimate bosses with special effects!
- ⛏️ **Underground Mining** - Tunnels, ores, depth progression!

### 🎯 HOW TO PLAY:
1. **Run the game**: `python run_money_craft.py`
2. **Movement**: Use WASD or Arrow keys to walk
3. **Mining**: Click blocks to mine them for wealth
4. **Enter Tunnels**: Press E when near tunnel entrances
5. **Shop**: Press S to open shop, click tabs to buy upgrades
6. **Villages**: Visit villages to see working villagers

### 🏘️ VILLAGERS:
- **Toolmakers** - Make pickaxes, swords, hammers, axes
- **Farmers** - Make bread, wheat, vegetables, fruit  
- **Miners** - Make coal, iron, copper, stone
- **Traders** - Make coins, gems, rare items, maps
- **Bakers** - Make bread, cake, pie, cookies

### 🛒️ SHOP FEATURES:
- **Tools Tab** - Upgrade mining tools
- **Weapons Tab** - Buy combat weapons
- **Items Tab** - Purchase special items
- **Bitcoin Tab** - Crypto mining upgrades

### 🌍 WORLD FEATURES:
- **21 Biomes** - Including Bitcoin mines, cosmic realms
- **29 Terrain Types** - Including 15 insane ore types
- **25 Structures** - Rainbow bridges, quantum portals, dragon lairs
- **Underground Tunnels** - Procedural mining tunnels with ores

### 💎 GEMS & WEALTH:
- **Ruby** - $100
- **Emerald** - $200  
- **Diamond** - $500
- **Bitcoin** - $777,777 (ultimate gem!)
- **15+ More Gem Types** - Including dragon, quantum, cosmic gems

### 🎮 CONTROLS:
- **WASD/Arrows** - Move character
- **Mouse Click** - Mine blocks
- **E** - Enter mining tunnel
- **S** - Open shop
- **ESC** - Pause/Menu

### 🚀 GETTING STARTED:
1. Make sure all files are in the `money craft` folder
2. Run `python run_money_craft.py`
3. Start mining and building your wealth!
4. Visit villages and explore the insane world!
5. Enter tunnels for underground mining!
6. Upgrade your tools in the shop!
