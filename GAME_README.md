# 🎮 REAL MINECRAFT - Complete Game Package

## 🚀 QUICK START

### Method 1: Run Launcher (Recommended)
```bash
python3 run_game.py
```

### Method 2: Run Direct
```bash
python3 real_minecraft.py
```

## 🎯 GAME FEATURES

### ⚔️ COMBAT SYSTEM
- **6 Weapons**: Wooden, Stone, Iron, Golden, Diamond Swords + Bow
- **Durability System**: Weapons wear out and break
- **Attack Mechanics**: Left-click to attack animals
- **Damage Values**: Each weapon has different attack power

### 🦊 ANIMALS
- **Regular Animals**: Cows, Pigs, Chickens, Sheep, Rabbits
- **SPECIAL FOXES**: Ralph (Orange, Speed 2.0) & Frank (Brown, Speed 1.8)
- **AI Movement**: Animals wander realistically on ground
- **Food Drops**: Hunt animals for food collection

### 🏗️ BUILDING SYSTEM
- **Block Preview**: See where blocks will place
- **Smart Placement**: Won't place on player/animals
- **20+ Block Types**: Dirt, Stone, Wood, Planks, Cobblestone, etc.
- **Inventory Management**: Hotbar + Full inventory system

### 🍖 SURVIVAL
- **NO HUNGER**: Play without survival pressure!
- **Health System**: Take damage from falls/combat
- **Food Collection**: Hunt animals for food (for fun)

### 🌍 WORLD
- **300x150 Blocks**: Massive explorable world
- **Terrain Generation**: Hills, caves, ores, structures
- **Day/Night Cycle**: Dynamic sky with sun/moon
- **3D-Style Graphics**: Shaded blocks for depth

## 🏭 CRAFTING RECIPES

### Weapons
- **Wooden Sword**: 2 Planks + 1 Stick
- **Stone Sword**: 2 Cobblestone + 1 Stick  
- **Iron Sword**: 2 Iron + 1 Stick
- **Golden Sword**: 2 Gold + 1 Stick
- **Diamond Sword**: 2 Diamond + 1 Stick
- **Bow**: 3 Sticks + 3 Wood

### Blocks
- **Planks**: 1 Wood
- **Sticks**: 1 Plank
- **Crafting Table**: 4 Planks
- **Furnace**: 8 Cobblestone
- **Torch**: 1 Stick + 1 Coal

## 🎯 TIPS

### Getting Started
1. **Gather Resources**: Left-click blocks to mine them
2. **Craft Weapons**: Press C, craft better swords
3. **Hunt Animals**: Left-click Ralph/Frank for valuable fox meat
4. **Build Shelter**: Right-click to place blocks
5. **Explore**: Discover caves, ores, and landscapes

### Combat Tips
- **Ralph is fastest** (Speed 2.0) - Harder to catch
- **Frank is fast too** (Speed 1.8) - Challenging prey
- **Use better weapons** for more damage
- **Fox meat restores 10 hunger** - Most valuable food!

### Building Tips
- **Use preview** to see placement before committing
- **Mix materials** for creative designs
- **Build vertically** for towers and structures
- **Use torches** for lighting and atmosphere

## 🔧 TROUBLESHOOTING

### Game Won't Start
```bash
# Install pygame manually
pip install pygame

# Or use launcher (auto-installs)
python3 run_game.py
```

### Performance Issues
- **Close other applications** for better FPS
- **Reduce render distance** (modify world size if needed)
- **Update graphics drivers** for better performance

### Controls Not Working
- **Click game window** to focus it
- **Check keyboard** connection
- **Restart game** if controls become unresponsive
