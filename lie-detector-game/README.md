# Professional Lie Detector Game

A realistic polygraph simulation game where you test your lie detection skills using advanced biometric readings.

## Features

### Realistic Polygraph System
- **Heart Rate Monitor**: Tracks BPM with realistic fluctuations
- **Blood Pressure Monitor**: Systolic pressure readings
- **Respiration Monitor**: Breathing rate analysis
- **Sweat Level Detection**: Galvanic skin response simulation
- **ECG/EKG Graph**: Real-time heart rate variability display
- **Stress Level Meter**: Overall stress indicator

### Advanced Gameplay
- **Question Database**: 24 carefully crafted questions with varying difficulty
- **Truth/Lie Detection**: Analyze statements to determine truthfulness
- **Confidence System**: Polygraph accuracy based on stress levels
- **Performance Tracking**: Score, accuracy, and detective rating
- **Professional Interface**: Authentic polygraph machine design

### Visual Effects
- **Real-time Graphs**: Animated ECG/EKG display
- **Vital Sign Monitors**: Medical-grade interface design
- **Analysis Animation**: Professional lie detection process
- **Screen Effects**: Visual feedback for correct/incorrect detections

## How to Play

### Objective
Test your ability to detect truth and lies by analyzing polygraph readings and making educated decisions.

### Controls
- **SPACE**: Start game / Continue to next question
- **T**: Mark statement as TRUTH
- **F**: Mark statement as FALSE
- **ESC**: Return to menu / Quit game

### Gameplay Mechanics

1. **Question Phase**: Read the statement carefully
2. **Analysis Phase**: Observe polygraph readings and stress indicators
3. **Decision Phase**: Press T for TRUTH or F for FALSE
4. **Result Phase**: See if your detection was correct
5. **Scoring**: Earn points based on difficulty and accuracy

### Scoring System
- **Easy Questions**: 100 points
- **Medium Questions**: 200 points
- **Hard Questions**: 300 points
- **Accuracy Bonus**: Higher accuracy increases confidence

### Detective Ratings
- **Expert Detective**: 80%+ accuracy
- **Skilled Analyst**: 60-79% accuracy
- **Developing Detective**: 40-59% accuracy
- **Needs Training**: Below 40% accuracy

## Game Strategy

### Tips for Success
1. **Monitor Stress Levels**: Higher stress may indicate deception
2. **Watch Heart Rate**: Spikes can signal lies
3. **Check Sweat Levels**: Increased sweating suggests stress
4. **Consider Difficulty**: Harder questions cause more stress
5. **Trust Your Instincts**: Balance readings with intuition

### Understanding the Polygraph
- **Baseline Readings**: Normal vital signs when calm
- **Stress Response**: Body reacts to lying with increased vitals
- **Confidence Factors**: High stress reduces detection accuracy
- **Random Elements**: Real polygraphs have inherent error rates
